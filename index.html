NPOらくらく損金算出機

「NPOらくらく損金算出機」は法人がNPO法人に寄付をした場合に損金算入できる金額の上限を計算するアプリです。 (※あくまでめやすとしてお使いください。)

Q1.法人の資本金はいくらですか？

寄付をする側の法人の資本金を入力してください。資本金がない法人の場合は0を入力してください。

円

Q2.所得金額はいくらですか？

寄付をする側の法人の所得金額を入力してください。

円

Q3.寄付先のNPO法人は認定NPO法人(もしくは特例認定NPO法人)ですか？

認定NPO法人とはNPO法人の中でも特に公益性が高く活動の適性が都道府県庁より認められたNPO法人です。寄付先が認定NPO法人の場合、通常のNPO法人に比べて大幅に損金算入額の上限が上がります。

  • 資本がある法人 => (期末資本金の額×0.25%＋所得金額×2.5%)×1/4
    資本がない法人 => 所得金額×1.25%
  • 資本がある法人 => (期末資本金の額×0.375%＋所得金額×6.25%)×1/2
    資本がない法人 => 所得金額×6.25%
損金算入上限額は0円です
